# House Price Prediction

**Author:** Eakansh Kshirsagar  
**Internship:** XYlofy (Week 1 Assignment)

## Overview
This project involves building a regression model to predict house prices based on property features (area, bedrooms, bathrooms, etc.). 

## Tasks Completed
1. **Data Exploration & Cleaning:** Handled missing values and applied one-hot encoding to categorical variables to prepare for machine learning.
2. **Model Building:** Trained, evaluated, and compared **Linear Regression** and **Random Forest Regressor** models.
3. **Visualizations:** Created histograms, correlation heatmaps, and actual vs. predicted scatter plots to understand the data distribution and model accuracy.
4. **Insights:** Identified total area and bathrooms as the top drivers of house prices, and calculated metrics (MAE, RMSE, R\u00b2) for both models. 

## Files Included
- `analysis.ipynb`: The complete Jupyter Notebook containing all Python code, outputs, and findings.
- `Housing.csv`: The original dataset used for training (545 rows, 13 columns).
- `summary.docx`: A concise 1-page business summary of the findings and model metrics.
- `charts/`: Folder containing all generated visualization PNGs.
